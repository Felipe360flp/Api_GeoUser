## Installation

```bash
$ npm install
```

## Installation Nest

$ npm i @nestjs/mapped-types

$npm i bcrypt

$import * as bcrypt from 'bcrypt';

## Installation Jwt

$npm i passport passport-jwt passport-local @nestjs/passport @nestjs/jwt

$npm i -D @types/passport-jwt @types/passport-local

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
